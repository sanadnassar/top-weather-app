<h1>☁️ SkyCast: Glassmorphism Weather App</h1>

A sleek, responsive weather dashboard built with vanilla JavaScript. This project fetches real-time meteorological data from the Visual Crossing API and presents it through a modern Glassmorphic interface.

<h1>🚀 Live Demo</h1>
https://sanadnassar.github.io/top-weather-app/

<h1>✨ Features</h1>
Real-time Data: Fetches current conditions, including temperature, humidity, and wind speed.

Dynamic Search: Search for any city globally with automated error handling for invalid locations.

Glassmorphic UI: A premium look using backdrop-filter and semi-transparent layers for a modern aesthetic.

Security Minded: Implements modular JS with .gitignore to protect sensitive API credentials during development.

<h1>🛠️ Tech Stack</h1>
HTML5: Semantic structure for accessibility.

CSS3: Advanced layouts using CSS Grid and Flexbox.

JavaScript (ES6+): Utilizes Async/Await, Fetch API, and ES Modules for clean, maintainable code.

API: Visual Crossing Timeline API.


Project built based on the "The Odin Project" javascript course requirements
